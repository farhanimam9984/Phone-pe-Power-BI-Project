Project Overview

-- This project is a data visualization and business intelligence dashboard built using Power BI, inspired by the digital payments ecosystem of PhonePe.
-- The dashboard provides interactive insights into user data, transaction patterns, and growth trends, helping businesses understand user behavior and make data-driven decisions.

Aim of the Project :- 
    -- The primary objective of this project is to:
    -- Analyze user data and transaction trends
    -- Identify growth patterns and user engagement
    -- Build an interactive dashboard for better decision-making
    -- Transform raw data into meaningful business insights
